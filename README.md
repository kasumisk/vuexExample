## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```

# pure mock data

``` bash
# dependencies  puer-mock
npm install -g puer-mock

# mock serve  serve port is 8000
npm run mock

# mock api config

edit _mockserver.json

```
